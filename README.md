Customer Support Ticket Analysis:-

Project Overview:
This project analyzes customer support tickets using a rule-based approach to classify issues, assign priorities, calculate SLA timelines, and generate managerial reports.

Dataset:
Customer Support Ticket Dataset sourced from Kaggle.

Problem Statement:
Support teams need a structured way to categorize tickets, prioritize urgent issues, and monitor SLA compliance.

Solution Approach:
- Data loading and exploration.
- Text cleaning and normalization.
- Rule-based issue classification.
- Priority assignment (P0â€“P3).
- SLA hour calculation and due-time estimation.
- Support manager summary report generation.

Tools & Technologies:
- Python
- Pandas
- Google Colab
- GitHub

Outputs:
- 'final_customer_support_report.csv': Ticket-level processed dataset.
- 'support_manager_summary.csv': Aggregated manager-level report.
